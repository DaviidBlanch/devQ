# devQ
Done by David Blanch

## About The Project
Website for technical interview questions and interviews for the developer world

## Getting Started
Install node.js:
```
$ npm install node.js
```

Run in dev mode:
```
$ npm run dev
```

### Built With
- [Node.js](https://nodejs.org/es/docs/)
- [Express](https://expressjs.com/)
- [MongoDB](https://www.mongodb.com/)
